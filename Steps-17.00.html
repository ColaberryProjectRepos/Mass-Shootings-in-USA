Create Victims Per Shooting by Weapon Type 17

Step 1. Copy and paste the By Shooters' Age Group visual

Step 2. Remove all the Data values from the Values well.

Step 3. Drag and drop Weapon Type to the Shared Axis, Victims Per Shooting                    measure to Column Values well and Average YOY measure to the Line                  Values well.



Step 4. Format the location and size of the visual



Steps

  1. Go to Format Pane and expand the General tab to display the options.
  2. Set the X Position to 1003.
  3. Set the Y Position to 600.
  4. Set the Width to 480.
  5. Set the Height to 400.

Step 4. Format the title of the visual


Steps

  1. Go to Title tab and expand the tab to display the options.
  2. Enter the title By Weapon Type into the Title Text Box.

Step 5. Turn the visual shadow on


Steps

In this step we will turn the visual shadow on to make it standout against the background.

  1. Click on the Visual.
  2. Go to Format Pane and turn the shadow on as shown in the image above.

This is a Microsoft documentation on how to create and format combo charts.

https://docs.microsoft.com/en-us/power-bi/visuals/power-bi-visualization-combo-chart
